# Java Quiz Application

## Project Description
The Java Quiz Application is a **console-based application** developed using **Core Java**.  
It allows users to answer multiple-choice questions and displays the final score at the end of the quiz.

This project is designed for **students** to understand basic Java concepts and object-oriented programming.
